# opengl
    - template project to get started with opengl on windows
    
# how to run
    1) cd into bin
    2) type: mingw32-make
    3) type: main.exe

# directory structure
    - bin
        - Makefile
        - glfw3.dll
        - main.exe
    - include
        -GLFW
        -KHR
        -glad
    - lib
        - libglfw3dll.a
    - src
        - components
        - glad.c
        - main.cpp
    README.md
